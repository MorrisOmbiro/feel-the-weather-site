Privacy Policy

Feel The Weather (Chrome extension)

Last updated: September 14, 2025

Overview

Feel The Weather replaces your browser’s New Tab page and shows local weather with animated visuals. We collect as little data as possible to deliver this single feature. We do not sell user data.

Data we collect & why

  • Approximate location (optional): If you grant the Geolocation permission, the extension reads your device’s approximate location to fetch local weather. Location is used in-memory to request weather and is not stored or shared by us.
  • Preferences: Temperature unit (°F/°C), manually entered city (if you choose not to use geolocation), and simple UI settings are stored locally using the Chrome storage API so the extension remembers your choices.

Third-party services

The extension requests weather data from a weather API provider (e.g., OpenWeather). Your browser makes HTTPS requests to the API which includes the city/coordinates needed to return weather. No personal identifiers are attached by the extension.

What we don’t do

  • No collection of names, emails, payment info, browsing history, keystrokes, or content of pages.
  • No selling or renting of data to third parties.
  • No remote execution of third-party code (we only fetch JSON weather data).
  • No tracking across sites for advertising or profiling.

Data storage & retention

Preferences are stored locally in your browser via the Chrome storage API and remain until you remove the extension or clear extension data. Location is not persisted by us; if you enter a city manually, that text is stored locally as a preference.

Chrome permissions used

  • geolocation (optional): to determine local weather if enabled.
  • storage: to save units (°F/°C), city, and UI settings.
  • alarms: to refresh weather on a schedule so data is current when opening a new tab.
  • host permissions: restricted to the weather API domain to retrieve weather JSON.

Your choices

  • Disable geolocation and enter a city manually in the extension’s settings.
  • Clear data via Chrome’s “Clear browsing data > Hosted app data” or remove the extension.

Children’s privacy

The extension is not directed to children under 13 and does not knowingly collect personal information from children.

Policy changes

If we make material changes, we will update this page and the “Last updated” date above.
